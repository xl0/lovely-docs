# lovely-docs-mcp

MCP server for lovely-docs

## Installation

```bash
npm install
npm run build
```

## Usage

Run the server:

```bash
node dist/index.js
```

Or use it as an MCP server in your MCP client configuration.

## Development

Watch mode for development:

```bash
npm run watch
```

## Available Tools

- `example_tool`: An example tool that echoes back the input
