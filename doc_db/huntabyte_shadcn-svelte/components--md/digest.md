## Available Components

Complete list of 70+ UI components available in shadcn-svelte:

**Layout & Structure**
- Accordion, Aspect Ratio, Breadcrumb, Card, Carousel, Collapsible, Data Table, Drawer, Empty, Resizable, Scroll Area, Separator, Sheet, Sidebar, Table, Tabs

**Forms & Input**
- Badge, Button, Button Group, Checkbox, Combobox, Field, Formsnap, Input, Input Group, Input OTP, Label, Native Select, Radio Group, Select, Slider, Switch, Textarea, Toggle, Toggle Group

**Dialogs & Overlays**
- Alert Dialog, Dialog, Context Menu, Dropdown Menu, Hover Card, Menubar, Navigation Menu, Popover, Tooltip

**Display & Feedback**
- Alert, Avatar, Calendar, Chart, Command, Item, Kbd, Pagination, Progress, Range Calendar, Skeleton, Sonner, Spinner, Typography

**Utilities**
- Spinner, Tooltip, Typography

Install components with:
```bash
npx shadcn-svelte@latest add <component> -y -o
```

Options:
- `-y`: skip confirmation prompt
- `-o`: overwrite existing files
