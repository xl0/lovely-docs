**Registry items** define components, styles, themes, blocks with JSON schema specifying `name`, `type`, `files`, `dependencies`, `registryDependencies`, `cssVars`, `css`. **Types**: `registry:block/component/ui/lib/hook/page/file/style/theme`. **Files** have `path`, `type`, optional `target` for installation location. **CSS variables** organized by scope (theme/light/dark). **CSS rules** support `@layer base/components`, `@utility`, `@keyframes`. **Build**: `npm run registry:build` generates JSON in `static/r/`. **Publish**: Deploy to public URL. **Install**: `npx shadcn-svelte@latest add <url> -y -o`. **Auth**: Implement token-based on server side.
