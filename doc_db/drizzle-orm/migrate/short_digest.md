## Purpose
Applies SQL migrations generated by `drizzle-kit generate` using code-first approach.

## Configuration
Requires `dialect` and database credentials via config file or CLI:
```ts
// drizzle.config.ts
export default defineConfig({
  dialect: "postgresql",
  schema: "./src/schema.ts",
  dbCredentials: {
    url: "postgresql://user:password@host:port/dbname"
  },
});
```
```shell
npx drizzle-kit migrate
```

Or via CLI:
```shell
npx drizzle-kit migrate --dialect=postgresql --url=postgresql://user:password@host:port/dbname
```

## Migrations log
Applied migrations stored in `__drizzle_migrations` table (customizable):
```ts
migrations: {
  table: 'my-migrations-table',
  schema: 'public', // PostgreSQL only
}
```

## Multiple configs
```shell
npx drizzle-kit migrate --config=drizzle-dev.config.ts
npx drizzle-kit migrate --config=drizzle-prod.config.ts
```
