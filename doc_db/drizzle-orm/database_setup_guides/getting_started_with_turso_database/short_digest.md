Install `drizzle-orm@beta`, `@tursodatabase/database`, `dotenv`, and dev dependencies `drizzle-kit@beta`, `tsx`. Set `DB_FILE_NAME` environment variable. Connect with `drizzle()` from `drizzle-orm/tursodatabase/database`. Define schema, setup `drizzle.config.ts` with SQLite dialect. Perform insert/select/update/delete operations using standard Drizzle API with `eq()` for WHERE clauses. Run with `tsx`.
