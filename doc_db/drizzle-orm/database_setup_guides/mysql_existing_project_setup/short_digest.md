## MySQL Existing Project Setup

1. Install mysql2 package
2. Set DATABASE_URL environment variable
3. Create drizzle.config.ts with mysql dialect
4. Introspect existing database schema
5. Transfer introspected schema to schema file
6. Connect using mysql2 driver
7. Query database with Drizzle ORM
8. Run with tsx
9. (Optional) Update schema, apply migrations, test new fields
