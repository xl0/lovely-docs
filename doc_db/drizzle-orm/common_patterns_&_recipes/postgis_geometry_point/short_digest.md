PostGIS extends PostgreSQL with geospatial support. Manually create extension with `CREATE EXTENSION postgis;`. Define geometry columns with `geometry('location', { type: 'point', mode: 'xy', srid: 4326 })` and add GIST spatial indexes. Insert points via object `{ x, y }`, tuple `[x, y]`, or raw SQL `ST_SetSRID(ST_MakePoint(x, y), srid)`. Query nearest locations using `<->` operator and `ST_Distance()`. Filter by rectangular area with `ST_Within()` and `ST_MakeEnvelope()`.
