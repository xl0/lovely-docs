## What is SvelteKit?

Framework for building web apps with Svelte. Similar to Next.js or Nuxt.

## Svelte vs SvelteKit

**Svelte**: UI component compiler that converts components to JavaScript and CSS.

**SvelteKit**: Complete framework adding routing, build optimization, offline support, configurable rendering (SSR/CSR/prerendering), image optimization, and HMR via Vite.
