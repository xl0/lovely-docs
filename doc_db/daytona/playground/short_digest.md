## Playground Overview

Interactive browser environment at https://app.daytona.io/dashboard/playground with three tabs operating on the same sandbox.

## Sandbox Tab
Configure sandboxes and generate SDK code:
- **Management**: select language, resources, lifecycle policies (creates sandbox on **Run**)
- **File System**: create/list/delete files and directories
- **Git Operations**: clone repos, get status, list branches
- **Process & Code Execution**: run shell commands and language-specific code snippets

## Terminal Tab
Web-based terminal on port 22222 for command-line access. Auto-creates sandbox if needed. Restricted to authenticated organization members.

## VNC Tab
Graphical desktop access with controls for:
- **Display**: get info, list windows
- **Keyboard**: hotkeys, keys, text input
- **Mouse**: click, drag, move, get position
- **Screenshot**: capture with format/scale/quality options
