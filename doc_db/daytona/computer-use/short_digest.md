## Computer Use

Programmatic desktop control in Daytona sandboxes: mouse (click, move, drag, scroll, position), keyboard (type, press, hotkey), screenshots (full/region, compressed), screen recording (start/stop/list/download), and display info (displays, windows).

Start/stop processes, get status, restart, view logs. All operations available in Python, TypeScript, Ruby, Go SDKs and REST API.

Recording directory configurable via `DAYTONA_RECORDINGS_DIR` env var. Recording dashboard in Daytona Dashboard UI.
